React 多入口配置方法

react官方给出的解决办法是直接使用webpack来配置多入口，原react不支持多入口。

配置方案

  1. 使用create-react-app 脚手架工具来创建react项目，打开react的eject.注意打开了就不能还原了

  2. 修改webpack配置 webpack.dev的entry修改，使用对象，并且多key-value扩展多入口

  3. 修改webpack配置 webpack.dev的output修改

  4. 修改webpack配置 HtmlWebpackPlugin生成多个html

  5. 修改webpack配置 webpackDevServer.config修改,保证url输出的正确

  6. 生产环境配置重复2步骤即可

配置完成后，每个入口都有自己对应的html，js，css等。
